# Linear Regression vs Random Forest Comparison

This project demonstrates the implementation and comparison of Linear Regression and Random Forest Regression models on a dataset. The workflow includes:

- Data Loading and Exploration
- Data Preprocessing
- Train-Test Split
- Linear Regression Model Training
- Random Forest Model Training
- Model Evaluation (R², MAE, RMSE)
- Performance Comparison
- Visualization using Matplotlib and Seaborn

The objective is to analyze the strengths and weaknesses of both algorithms and determine which model performs better on the given dataset.
